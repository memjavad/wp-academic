## 2024-04-22 - Ensure layout is array if stored as JSON string **Learning:** When validating variables decoded from JSON, especially inputs going into foreach loops or layout builders, fallback to default types instead of just skipping processing. `is_array( $decoded ) ? $decoded : []` is better than just a conditional re-assignment. **Action:** Always enforce a type guarantee at boundaries where JSON is decoded and directly manipulated as array.
