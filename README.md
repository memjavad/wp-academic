# WP Academic Post Enhanced

[![License: GPL v2](https://img.shields.io/badge/License-GPL%20v2-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-3.9.3-green.svg)](wp-academic-post-enhanced.php)

A professional, research-oriented WordPress plugin designed to transform standard blog posts into academic-grade articles. Optimized for SEO, Large Language Models (LLMs), and superior reader experience.

---

## 🚀 Key Features

### 🎓 Academic Citation System
Empower your readers to cite your work easily. Supports multiple popular formats:
- **Formats:** APA, MLA, Chicago, Harvard, IEEE, AMA.
- **Export:** Download `.RIS` files compatible with Zotero, Mendeley, and EndNote.
- **UI:** Tabbed interface with one-click copy functionality.

### 📚 Learning Management System (LMS)
Full-featured course management module for academic training.
- **Courses & Lessons:** Structured curriculum builder.
- **Quizzes:** Integrated assessment tools.
- **Progress Tracking:** Interactive progress bars and lesson completion logic.

### 🤖 Field News AI
Leverage AI to generate research-backed news articles automatically.
- **Gemini Integration:** Uses Google Gemini for high-quality content generation.
- **Academic Source Sync:** Integrates with Scopus APIs to ensure factual accuracy.
- **Auto-Scheduling:** Set it and forget it news generation.

### 🔍 Advanced SEO & Metadata
Help search engines and AI assistants understand your academic content.
- **JSON-LD Schema:** Automatically adds Article and BlogPosting schema.
- **Table of Contents (TOC):** SEO-friendly, clickable TOC with smooth scrolling.
- **Sitemap Optimization:** Enhanced sitemaps for deep content discovery.

### ⚡ Performance & Optimization
Keep your academic site fast and responsive.
- **Critical CSS:** Automatically generates and injects critical styles.
- **Asset Manager:** Conditional loading of CSS/JS files to reduce bloat.
- **DB Optimizer:** Keep your WordPress database lean and performant.

---

## 🛠 Technical Details

- **Language:** Modern PHP (OOP Architecture).
- **Design System:** Unified CSS Variable-driven UI for visual consistency.
- **No Dependencies:** Runs entirely on Vanilla JavaScript & CSS (No heavy libraries).
- **Security:** Built-in sanitization, escaping, and nonce verification.

---

## 📦 Installation

1. Clone or download this repository into your `/wp-content/plugins/` directory.
2. Activate the plugin via the **Plugins** menu in WordPress.
3. Access the **Academic Post** dashboard to configure your modules.
