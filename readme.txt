A comprehensive suite of tools to enhance WordPress posts for academic and research-oriented content.

WP Academic Post Enhanced is a modular, research-oriented plugin designed to transform standard blog posts into professional, academic-grade articles. It focuses on improving Search Engine Optimization (SEO), Large Language Models (LLMs) comprehension, and superior reading experiences.

**Key Features:**

*   **Smart Citations:** Automatically generate citations in APA, MLA, Chicago, Harvard, IEEE, and AMA formats. Includes a downloadable .RIS file for reference managers.
*   **Course Management:** A built-in LMS for academic courses, including quizzes and progress tracking.
*   **Field News AI:** Generate research-backed news stories using LLMs and academic data sources.
*   **PDF Download:** Generate clean, print-ready PDF versions of your articles with customizable cover pages and styling.
*   **Table of Contents:** Automatic, SEO-friendly Table of Contents generation.
*   **Schema Markup:** Adds rich JSON-LD schema (Article, BlogPosting) to help search engines and LLMs understand your content.
*   **Privacy-Focused Analytics:** Built-in visitor tracking that respects user privacy (no external cookies).
*   **Social Sharing:** Customizable social sharing buttons with support for mobile sticky bars.
*   **Performance Tools:** Includes critical CSS generation, asset manager, and database optimization features.
*   **Reading Tools:** Estimated reading time and progress bar.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/wp-academic-post-enhanced` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Navigate to "Academic Post" in the admin sidebar to configure the modules.

== Frequently Asked Questions ==

= Does this plugin require any external services? =
No, all features including analytics and PDF generation run entirely on your server.

= Can I disable features I don't use? =
Yes, the plugin is modular. You can enable or disable specific features (like Ads, Analytics, or Social Sharing) from the main dashboard.
